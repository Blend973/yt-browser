# yt-browser
A simple script to watch youtube from your terminal. Just lean and clean clutter free experience.
## Features
1. Interactive Menu: Text-based ui for video selection using `fzf` or `rofi`.
2. Download Option: Download videos easily using interactive UI.
3. Lean and Clean: No extra stuff just simple interface and functionality.
4. Playback Mode: Play videos or just use audio-only mode to listen to your     chosen music.
5. Autoplay: You can autoplay videos or audios using autoplay mode. There's     two different autoplay mode. One to play related contents and the other      for playlist. You can play all search results as playlist.
6. Image and Text preview: Thumbnail previews and video metadata like           duration, channel etc. are also shown during selection.
7. Easy Configuration: You can tweak video quality, selector, and other         things through config file.
## Dependencies
To run this script without any error you should have these in your system:
1. Kitty Terminal(Recommended)
2. jq
3. curl
4. mpv
5. ffmpeg
6. yt-dlp
7. fzf
